# Filter Setup — Admin Instructions

Go to: Shopify Admin → Online Store → Navigation → Search & Discovery → Filters

Configure filter groups in this exact order:

1. Availability (existing — keep)
2. Product type → Category
3. Tag: ports-* → Port count
4. Tag: wattage-* → Wattage
5. Tag: connection-* → Connection
6. Tag: compatible-* → Works with
7. Price (existing — keep, move to bottom)

Apply to all collections, especially:

- hubs-adapters
- chargers-power
- accessories
