# Page Hero Setup Guide

This theme now uses the `Page hero` section to provide a dark backdrop for the overlay navigation on non-home templates.

## Where To Edit In Shopify Admin

1. Open **Online Store → Themes → Customize**.
2. Use the template picker to switch between templates (product, collection, support pages, FAQ, compare, cart, search, 404).
3. Select the top `Page hero` section.
4. Configure:
   - **Background image**
   - **Background fallback color**
   - **Kicker**
   - **Heading**
   - **Sub-heading**
   - **Hero height (px)**
   - **Dark overlay opacity**

## Recommended Image Specs

- Desktop: **1920 x 600** minimum
- Safe target: **2400 x 900** for sharper large-screen rendering
- Keep important subject matter away from the very top edge so navigation stays readable.

## Overlay Navigation Readability Note

The overlay nav uses white text/logo at rest. For readability, the top header zone should be dark:

- Aim for strong dark coverage in the top ~108px of the hero.
- Increase overlay opacity if needed.
- If an image is too bright, use a darker image or increase overlay opacity.

## Suggested Heights

- Product / FAQ / Compare: **320px**
- Collection / Support: **360px**
- Cart / Search / 404: **280px**

Adjust by template as needed for content density and visual rhythm.
